Étape indispensable pour continuer :

1. Cliquez sur le bouton ci-dessous pour ouvrir les données de propagation dans un nouvel onglet.
2. Dans la fenêtre qui s'ouvre, faites clic droit sur la page puis « Enregistrer sous… ».

4. Revenez ici et importez ce fichier grâce au bouton « Importer propagation » dans la colonne de gauche.

Astuce : Ne vous souciez pas du format, l’application va lire directement le fichier JSON – votre vie privée et sécurité sont garanties (le fichier n’est jamais envoyé sur internet).
